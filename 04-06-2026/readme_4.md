## What is this project?

## Files

### `ai_engineer_career_roadmap.html`

A personalised, one-page career roadmap generated by Claude based on a real conversation.

**Inputs used:**
- Situation: Working Professional
- Skills: Python, Web Development, Data Analysis
- Goal: Become an AI Engineer
- Timeline: 6 months

**What it contains:**
- Current position snapshot
- Target goal with motivating one-liner
- Skill gap analysis with visual bars (have vs need)
- 3-phase learning plan with specific resources
- 4 suggested portfolio projects
- Networking strategy (LinkedIn, communities, open source)
- Month-by-month milestones (6 months)
- 5 immediate next actions for this week

This file is screenshot-ready and export-ready for LinkedIn sharing.

---

### `task_4_04_06_2026.html`

A fully working static web app — a form-based Career Roadmap Generator.

**How it works:**
1. User fills in 4 fields — situation, skills, goal, timeline
2. Clicks "Generate My Roadmap →"
3. Gets a complete, dynamically generated roadmap on screen
4. Can download it as a PDF or start over

**What makes it dynamic:**
- Pure HTML + CSS + JavaScript — no backend, no API calls
- JavaScript logic maps each goal + situation + skill combination to specific content
- Skill gap analysis calculates a match score and animates a progress bar
- Monthly milestones scale automatically based on chosen timeline
- Print-optimised CSS so the PDF output looks clean

**Fields in the form:**
- Current Situation — dropdown (Student, Working Professional, Freelancer, Founder, Career Switcher)
- Current Skills — multi-select checkboxes (Python, Data Analysis, Web Dev, ML basics, SQL, Cloud, Math/Stats)
- Target Goal — dropdown (Get a Job, Internship, Data Scientist, AI Engineer, Start a Business, Grow on LinkedIn)
- Target Timeline — radio buttons (3 months, 6 months, 1 year, 2 years)
