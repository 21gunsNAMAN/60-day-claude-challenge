Most impressive achievement to lead with

At HCLTech, I designed and built a multi-tenant WhatsApp communication platform from scratch using Laravel. The system had to serve multiple business clients from a single codebase — meaning I had to implement tenant isolation, manage API rate limits from Meta's WhatsApp Cloud API, and build a template management system, all in a 3-month internship window. That project taught me more about production system design than anything else I've done.

Key stories to practice (STAR format)

Story 1 — Technical problem solving

Building the multi-tenant architecture at HCLTech: what was the challenge, how did you decide on the approach, what trade-offs did you make, what was the outcome?

Story 2 — Working under pressure

SIH2024: How your team scoped the AI detection problem, divided responsibilities, integrated Django + YOLOv9s + LSTM in a hackathon timeline, and what specifically you contributed.

Story 3 — Ownership & delivery

Indian Express CMS: how you independently built an editorial CMS for a media company's live workflow, handled the feedback loop, and delivered a working product.

Likely interview questions — be ready for these

Explain multi-tenancy and how you implemented it in Laravel.

How does the WhatsApp Cloud API work? What were the key integration challenges?

Walk me through a project where you had to make an architectural decision.

What's the difference between CodeIgniter and Laravel? When would you use each?

How did you handle security in your PHP applications (SQL injection, CSRF, etc.)?
