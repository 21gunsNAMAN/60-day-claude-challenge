## Key Learnings

1. **Claude generates recruiter-ready copy in one shot** when given a detailed resume — no generic filler, actual project names and company names are woven in.
2. **Skill gap analysis is the most underrated section** — it gave me a specific 6-month roadmap rather than vague advice like "learn cloud."
3. **WhatsApp Cloud API experience is a genuine differentiator** at the fresher level — Claude flagged it immediately as rare and high-demand.
4. **Personal branding starts with positioning, not aesthetics** — the headline and UVP Claude generated are more specific than anything I had written before.
5. **Prompt quality = output quality** — providing a detailed prompt with clear section requirements made the output immediately usable.

## How to Replicate This

1. Open [claude.ai](https://claude.ai)
2. Upload your latest resume (PDF)
3. Set effort level to Low
4. Paste the prompt template from the challenge
5. Optionally add a target job description for more personalised output
6. Review and save each section
7. Add to your GitHub repo as `day12.md`
