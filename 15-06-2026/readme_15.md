## 💡 Key Learnings

1. **Structured inputs lead to structured outputs** – When I provided exact birth time and accuracy, Claude automatically calculated planetary positions and dashas. Vagueness would have produced generic text.
2. **Multi‑step reasoning is crucial** – The prompt forces Claude to first analyze the chart, then life patterns, then forecasts. Each section builds on the previous one, avoiding contradictions.
3. **Tables improve readability** – The 5‑year forecast table is far more useful than paragraphs. Claude follows “use tables wherever possible” very well.
4. **Remedies must be justified** – Claude only suggested gemstones after checking planetary strength. Without that rule, it might recommend everything.
5. **Effort level matters** – On “Low”, Claude produced concise ~2000 words. On “High”, it would likely exceed context length. “Low” was perfect for a 45‑min exercise.
