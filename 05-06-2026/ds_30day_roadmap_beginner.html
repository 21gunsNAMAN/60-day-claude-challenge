Week 1 project

Student performance data cleaner

Download the "Student Performance" dataset from Kaggle. Clean it, handle missing values, and produce a summary statistics report using pandas. Save as a Jupyter notebook.

Your HTML/CSS/JS skills will shine here — matplotlib and Seaborn charts can be exported to interactive web dashboards later!

Week 2 project

COVID-19 or IPL data EDA report

Pick a dataset (COVID stats, IPL cricket, or Zomato). Do a full EDA: 5+ charts, correlation heatmap, key findings written in markdown cells. Share on GitHub with a README.

This is the big week! Machine learning can feel overwhelming — focus on understanding the "why" before the math. Scikit-learn makes it surprisingly approachable.

Week 3 project

House price or spam classifier

Choose: predict house prices (regression) or classify emails as spam (classification). Use scikit-learn end-to-end — preprocessing, training, evaluation. Document every step in Jupyter.

Your web skills are a superpower here — use Streamlit to turn your ML model into a live web app. Employers love seeing deployed projects!

Capstone project (Days 28–30)

Live data science web app — your portfolio piece

Pick a topic you care about (sports, movies, health, finance). Collect/clean data, do EDA, train an ML model, and deploy a Streamlit app with live predictions. Share the GitHub link + app URL on LinkedIn.

Your outcome after 30 days

3 GitHub projects, 1 live deployed ML app, hands-on experience with pandas, scikit-learn, and Streamlit — enough to apply for data science internships and freelance data projects.
